[프로젝트 개요]
- 미국 정부 조달 사이트(SAM.gov) API 연동을 통한 주한미군 입찰 정보 자동 수집 및 분석 시스템 구축

[프로젝트 배경 및 목표]
- 미국 정부의 공식 입찰 사이트인 SAM.gov에서 제공하는 데이터를 활용하여, 주한미군(USFK) 관련 입찰 공고를 자동으로 수집 및 분석하는 웹 서비스 개발을 목표로 합니다. 본 프로젝트를 통해 조건에 맞는 입찰 공고를 선제적으로 발굴하고, 과거 데이터를 기반으로 한 분석을 통해 경쟁력 있는 사업 전략을 수립하고자 합니다. 단순 공고 조회를 넘어, 데이터 기반의 선제적 입찰 대응 시스템을 구축하는 것이 핵심입니다.

[과업 범위]
1. 수행 범위
- 상세 기획: 요구 사항 정의, 기능 정의, 화면 설계, 정책 설계
- UI/UX 디자인
- 프런트엔드/Client 개발 (PC Web - 반응형 지원)
- 백엔드 개발
- 서버/DB/인프라 구성

2. 상세 기능 요구 사항
2-1. 입찰 공고 자동 수집 및 알림
- SAM.gov 사이트의 입찰 정보를 API를 통해 주기적으로 스캔 및 수집
- 사전 공지(Presolicitation), 신규 공고, 변경 공고, 마감 임박 공고 등 상태별 구분
- 조건에 부합하는 공고 발생 시 지정된 담당자에게 이메일, 문자, 메신저 등으로 실시간 알림 전송
2-2. 맞춤 입찰 정보 필터링 및 선별
- 카테고리, 키워드, NAICS 코드 등 다양한 조건 기반의 필터링 기능
- 설정된 필터 조건에 따라 참여 가능성이 높은 입찰 리스트 자동 생성
2-3. 과거 입찰 데이터 분석
- 과거 낙찰(Award) 데이터를 수집하여 경쟁사, 낙찰 금액, 발주 패턴 등 분석
- 데이터 기반의 유망 사업 분야 도출 및 경쟁사 동향 파악 기능
2-4. 검색 및 데이터 관리
- 국가, 기관, 기간 등 상세 조건별 입찰 정보 검색 기능
- 검색 결과를 시스템 내 데이터베이스에 저장하고 관리
- 수집 및 분석된 데이터를 엑셀 파일로 다운로드하는 기능
2-5. 관리자 기능
- 사용자 계정 및 권한 관리
- 입찰 정보 필터링 키워드 및 조건 설정
- 알림 수신 채널 및 주기 등 시스템 환경설정
- 데이터 수집 현황 및 시스템 로그 모니터링

[지원 디바이스]
- PC Web (반응형 디자인 적용)

[기술 스택]
- 프로젝트에 가장 적합하다고 판단되는 기술 스택을 자유롭게 제안해 주시기 바랍니다. (단, 데이터 수집 방식은 안정성을 고려하여 API 연동을 우선적으로 검토)

[지원 자격 및 우대 사항]
1. 지원 자격
- 외부 API 연동 및 데이터 처리 프로젝트 경험을 보유한 팀 또는 기업
- 공공 데이터 또는 정부 조달 관련 시스템 개발 경험 보유
2. 우대 사항
- 대용량 데이터 수집(크롤링/API) 및 분석 플랫폼 구축 경험
- 유사 정부 지원 사업 수행 경험
